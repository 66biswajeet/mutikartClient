Place carousel images in this folder and name them `slide-1.jpg`, `slide-2.jpg`, ... `slide-6.jpg`.

They will be served at `/carousel/slide-1.jpg` etc. If an image is missing the carousel will display a grey placeholder instead.

Recommended sizes:

- 1200x400 (desktop)
- Also add mobile variants if desired (e.g., `slide-1-mobile.jpg`) and modify the component to load them.

Supported formats: `.jpg`, `.png`, `.webp`.
